Consulta de Ventas

GET
/api/ventas

Obtiene datos de ventas según los filtros especificados.

Parámetros:

  • fechaInicio - Fecha de inicio en formato YYYY-MM-DD
  • fechaFin - Fecha de fin en formato YYYY-MM-DD
  • sucursal - ID de sucursal (por defecto: 211)

Ejemplo de solicitud:

/api/ventas?fechaInicio=2025-01-01&fechaFin=2025-01-31&sucursal=211
Probar API

Exportar Ventas a Excel

GET
/api/ventas/excel

Exporta datos de ventas a un archivo Excel según los filtros especificados.

Parámetros:

  • fechaInicio - Fecha de inicio en formato YYYY-MM-DD
  • fechaFin - Fecha de fin en formato YYYY-MM-DD
  • sucursal - ID de sucursal (por defecto: 211)

Ejemplo de solicitud:

/api/ventas/excel?fechaInicio=2025-01-01&fechaFin=2025-01-31&sucursal=211

Consultas MySQL

POST
/api/mysql/query

Ejecuta una consulta personalizada en MySQL.

Ejemplo de solicitud básica:

{
  "query": "SELECT * FROM productos LIMIT 10",
  "args": []
}

Ejemplo de solicitud avanzada (Inventario 2025):

{
  "query": "SELECT COD_ART AS Codigo_Producto, Marca AS MARCA, SubFamilia AS CATEGORIA, saldos.NomDetSubFam AS DIMENSIONES, ZET_ART AS Zeta, ANIO_PRO AS Anio_Produccion, DES_INT AS Nombre_Producto, UNI_CAJ AS Packing, SUM(CAN_ING) AS Cantidad_Total, SUM(CIF_UNI * CAN_ING) / SUM(CAN_ING) AS PromedioPonderadoCostoCIF, cos_uni AS Costo_Real, MAX(fec_ing) AS Fecha_Ingreso, DATEDIFF(CURDATE(), (SELECT MAX(fec_ing) FROM saldos AS s2 WHERE s2.COD_ART = saldos.COD_ART AND s2.ZET_ART = saldos.ZET_ART AND s2.ANIO_PRO = saldos.ANIO_PRO)) AS Dias_Desde_Ingreso, (SELECT SUM(s2.CIF_UNI * s2.CAN_ING) / SUM(s2.CAN_ING) FROM saldos s2 WHERE s2.COD_ART = saldos.COD_ART AND s2.ANIO_PRO = saldos.ANIO_PRO) AS GlobalPromedioCostoCIF FROM saldos WHERE CAST(ANIO_PRO AS SIGNED) = 2025 GROUP BY COD_ART, Marca, SubFamilia, saldos.NomDetSubFam, ZET_ART, ANIO_PRO, DES_INT, UNI_CAJ, cos_uni ORDER BY COD_ART, ANIO_PRO",
  "args": []
}

  "query": "SELECT TOP 100 * FROM VENTA_BOLETA",

  "filename": "ventas.xlsx"
}
